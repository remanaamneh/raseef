אתר תפריט דיגיטלי עם תמונות לכל מוצר - ערבית ועברית.

## 🚀 הפעלה

זהו אתר סטטי (HTML/CSS/JS) שניתן להריץ ישירות בדפדפן או להעלות ל-GitHub Pages.

## 📱 QR Code

לפתיחת האתר מכל מחשב אחר, סרוק את ה-QR Code דרך הכפתור "📱 QR Code" באתר.

## 🌐 העלאה ל-GitHub Pages

1. העלה את כל הקבצים ל-repository ב-GitHub
2. לך ל-Settings > Pages
3. בחר את ה-branch הראשי (main/master)
4. בחר את התיקייה `/ (root)`
5. לחץ על Save
6. האתר יהיה זמין בכתובת: `https://remanaamneh.github.io/raseef/`
